#  五張撲克牌 (Five-Card Poker) - 視窗程式設計作業

**開發環境：** Visual Studio 2022 (C# Windows Forms)

---

##  專案簡介
本專案為一個完整的五張撲克牌投注遊戲。玩家可以輸入押注金額，並透過「發牌 ➔ 換牌 ➔ 判斷牌型」的標準撲克牌流程來贏取獎金。除基礎要求外，本作特別強化了「聲光效果」與「遊戲沉浸感」，並實作了極度完善的防呆機制與隱藏彩蛋。

##  遊玩說明
1. **下注：** 於「押注金額」欄位輸入金額（防呆機制會阻擋無效輸入或資金不足），按下「押注」。
2. **發牌：** 按下「發牌」按鈕，系統將自動洗牌並發出 5 張覆蓋的牌（附帶發牌音效與動畫延遲）。
3. **換牌 (單次機會)：** 點擊畫面上想更換的牌（牌面會蓋上），選好後點擊「換牌」抽取新牌。
4. **判斷牌型：** 按下「判斷牌型」，系統會自動計算中獎倍率、結算獎金，並播放對應的勝利/失敗音效。

##  創意項目

為了提升遊戲體驗與流暢度，本專案實作了以下額外功能：

### 1.  雙聲道音效與背景音樂 (BGM)
* **背景音樂：** 使用 `WindowsMediaPlayer` 獨立播放賭場風格 BGM，無限循環且不被音效干擾。
* **動態音效：** 使用 `SoundPlayer` 配合 `Task.Delay`，在「洗牌發牌」、「贏得獎金」與「輸錢」時播放對應的音效。

### 2.  破產與貸款紓困系統 (隱藏劇情)
* 當玩家將總資金輸光至 $0 時，遊戲會進入「破產模式」：
  * 背景音樂瞬間切換為悲慘的專屬 BGM。
  * 原本的押注按鈕被鎖死，取而代之的是隱藏的 **「申請紓困金」** 按鈕與負債計分板。
* **還款機制：** 貸款後若玩家奮發圖強，將資金贏回超過 $3,000 元，畫面會解鎖「還款」按鈕，讓玩家清償債務。
